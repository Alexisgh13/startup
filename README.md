# Gollum

Requires Python3.6+

## RUN IT

### From sources

Create virtualenv :   
`virtualenv -p python3 venv`

Activate it :  
`source venv/bin/activate`

Install dependencies :  
`pip install -r requirements.txr`

Start Flask server :  
`FLASK_APP=app.py flask run`

Then your server is available at :  
`http://localhost:5000`
